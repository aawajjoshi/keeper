# Keeper

## About

Keeper is a quick sticky note taking application that runs on your browser.

## Installing/Running

1. Install dependencies

   ```bash
   npm install
   ```

2. Fire up the dev server

   ```bash
   npm start
   ```

## Built with

- [React JSX](https://reactjs.org/docs/introducing-jsx.html)
